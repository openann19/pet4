# Keyboard Shortcuts

This document lists all available keyboard shortcuts in the application.

## General Shortcuts

### Navigation
- **Tab** - Move to next focusable element
- **Shift + Tab** - Move to previous focusable element
- **Escape** - Close modals, dialogs, or cancel current action
- **Enter** - Activate button or submit form
- **Space** - Activate button or toggle checkbox

### File Operations
- **Ctrl/Cmd + Z** - Undo last action (if supported)
- **Ctrl/Cmd + S** - Save (if supported)

## Chat Shortcuts

### Message Input
- **Enter** - Send message
- **Shift + Enter** - New line in message input
- **Ctrl/Cmd + K** - Open command palette (if supported)

### Message Navigation
- **Arrow Up** - Move to previous message
- **Arrow Down** - Move to next message
- **Arrow Left** - Move to previous conversation
- **Arrow Right** - Move to next conversation

## Drag and Drop Alternatives

### File Upload
- **Space** or **Enter** - Open file picker (keyboard alternative to drag and drop)
- **Arrow Keys** - Navigate file picker (if supported)
- **Escape** - Cancel file selection

## Accessibility

### Screen Reader
- **Ctrl/Cmd + Alt + Z** - Open keyboard shortcuts help (if supported)
- **Ctrl/Cmd + Alt + H** - Open accessibility help (if supported)

## Browser Shortcuts

The following browser shortcuts are not overridden and work as expected:
- **F1** - Help
- **F5** - Refresh
- **F12** - Developer Tools
- **Ctrl/Cmd + W** - Close tab
- **Ctrl/Cmd + T** - New tab
- **Ctrl/Cmd + R** - Refresh
- **Ctrl/Cmd + L** - Address bar

## Notes

- All keyboard shortcuts are customizable through the keyboard shortcuts registry
- Keyboard shortcuts can be disabled through user preferences
- Screen reader users can access keyboard shortcuts help through the accessibility menu
- Keyboard shortcuts are documented in the application's accessibility features

## WCAG 2.2 Compliance

All keyboard shortcuts comply with WCAG 2.2 AAA requirements:
- Keyboard shortcuts are documented
- Keyboard shortcuts do not conflict with browser shortcuts
- Keyboard shortcuts are discoverable through the help modal
- Keyboard shortcuts are accessible to screen reader users
