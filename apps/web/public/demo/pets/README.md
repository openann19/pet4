# Demo Pet Assets

Place the following pet images in this directory:

- `luna.jpg` - Background image for Luna (Golden Retriever)
- `luna-avatar.jpg` - Avatar image for Luna
- `milo.jpg` - Background image for Milo (British Shorthair)
- `milo-avatar.jpg` - Avatar image for Milo
- `max.jpg` - Background image for Max (Mixed breed)
- `max-avatar.jpg` - Avatar image for Max
- `fallback-avatar.jpg` - Fallback avatar image (optional)

If images are missing, the demo will gracefully fallback to `/demo/gradient.jpg` for backgrounds and `/demo/pets/fallback-avatar.jpg` for avatars.
